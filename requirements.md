## Scope

- Inventory management
- Inventory updates
- Purchases records
- Point of sale (scan code, or enter a code to modify the inventory of a product)

## Objective

The final goal is to have precise control over inputs and outputs to determine sales strategies appropriate for each season and thus achieve better cash flow

## Platform requirements

- Must run on iOS
- Must run on Android
- Must run on Web
